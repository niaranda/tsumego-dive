The project

TsumegoDive is the product of a bachelor's degree final year project by UPM (Universidad Polit√©cnica de Madrid) student Nina Aranda. All code developed for the project is available at GitHub under a GNU General Public License, along with more information about the authors.

This project aims to offer a new kind of perspective on tsumego solving, by helping the user explore the tsumego at their own pace in a fully interactive way. TsumegoDive gives recommendations based on a neural network which has been trained by analyzing over 16000 tsumegos.



How to use it

You can insert your tsumego by using the edit buttons or uploading an SGF file. Clicking on a placed stone again while in edit mode will remove it from the board. Once you are done, you will be required to choose the color of the first move.

Clicking Start will allow you to start looking for the solution. A game tree will appear on the screen which is automatically filled as you explore and can be used to navigate through your search by using your keyboard's arrow keys or the corresponding buttons. You can also mark movements as correct or wrong to guide your search. To remove correct or wrong markings you may also mark a movement as unknown through the question mark button.

When in doubt, use the Dive button or your keyboard's space bar to receive a recommendation. You can receive more recommendations by using this function repetitively.

If you want to save your search at any moment, use the Save as sgf button to download the game tree in SGF format.
